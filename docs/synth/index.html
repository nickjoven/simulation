synth · framework as a signal chain

Same picture as the shapes catalog, but the framework is wired up as four modules in series. One signal — N phases on S¹ — flows left to right; every module has knobs (under manipulation) and a scope (under view). The Kuramoto integrator runs continuously; turn knobs and watch the order parameter R re-settle.
